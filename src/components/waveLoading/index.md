## 水波进度组件 ##


## 使用方法 ##
* 通过 import { WaveLoading } from "@nestsoft/backtop"; 引入模块


## 组件配置 ##

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| progress | 进度百分百，必选 | string | - |
| bgColor | 背景色，可选 | string | "rgba(0,0,0,.7)" |
| waveColor | 进度颜色，可选 | string | "#333" |
| color | 图片url，可选 | string | "#fff" |
