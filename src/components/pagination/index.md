## 分页组件 ##


## 使用方法 ##
* 通过 import { Pagination } from "@nestsoft/backtop"; 引入模块


## 组件配置 ##

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| count | 数据总量，必填 | number | - |
| pageSize | 单页数量，必填 | number | - |
| offset | 当前页码，必填 | number | - |
| clickPage | (num: number) => void，必填 | string | - |
