## 组件库 ##


## 技术细节 ##
* 开发语言：Typescript

    
## 依赖 ##
* [react]：react框架
* 使用iconfont图标，请在终端项目中设置字体文件，fontFamily name为iconfont


## Install ## 
* yarn add https://github.com/yetingmao/React-Components.git




## 文件结构 ##
* src/  `源代码目录`
  * component `组件`
  * index.tsx `项目入口文件`
* .gitignore  `git配置文件`
* info.md  `记录一些信息`
* package.json  `包配置信息`
* readme.md  `此文件`
* tsconfig.json  `typescript编译配置`
* yarn.lock  `yarn生成的包依赖信息`
